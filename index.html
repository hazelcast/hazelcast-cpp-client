Hazelcast C++ Client

The Hazelcast C++ Client brings the performance and scalability of the Hazelcast in-memory computing platform to the Microsoft C++ ecosystem. With the Hazelcast C++ Client, you can dynamically scale your C++ workloads, achieve high read performance, and work seamlessly with Hazelcast data structures including distributed maps, queues, topics, and more—all secured with enterprise-grade SSL and mutual authentication.

This site provides comprehensive documentation, including examples, guides, FAQs, and a complete API reference generated from source-level comments and enriched with additional context.

The Hazelcast C++ Client is open source under the Apache License 2.0, and we welcome community contributions. The project is hosted on GitHub at hazelcast/hazelcast-cpp-client, and this documentation includes instructions on accessing the source code, building the client, collaborating with the community, and contributing improvements.

As with any documentation site, this one continues to evolve. If you find anything that could be improved, we’d love to hear from you. Contributions to the documentation are also welcome.
