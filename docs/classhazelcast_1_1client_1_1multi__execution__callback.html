Detailed Description

template<typename V>
class hazelcast::client::multi_execution_callback< V >

MultiExecutionCallback provides notification for when an execution is completed on each member that a task is submitted to.

After all executions are completed on all submitted members, the onComplete(std::unordered_map) method is called with a map of all results.
